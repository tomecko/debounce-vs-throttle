debounce vs throttle

confused with debounce and throttle?

click anywhere on this page and observe the difference,
try clicking multiple times in a short timespan for the best effect
(the demo uses rxjs v6 under the hood)

all clicks

just all clicks, no operation applied

.debounceTime(2000)

emits only after 2 seconds has passed without another emission (docs)
example: debouncing search input changes so expensive searches are not run while a user is typing

.throttleTime(2000)

emits and then ignores subsequent emissions for 2 seconds (docs)
example: throttling "refresh data" action to avoid multiple requests sent at the same time
